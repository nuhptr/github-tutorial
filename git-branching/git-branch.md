### Git Branch

```sh
Melihat branch saat ini :
$ git branch --show / --show-current

Melihat Semua Branch yang Ada di Repository Kita :
$ git branch --list
$ git branch

Membuat Branch Baru :
$ git branch namabranchbaru

- pindah ke branch lain
// git switch namabranch
// git checkout namabranchbaru
```
