### Clone

```sh
# Melakukan clone
- untuk melakukan clone, kita bisa gunkana perintah:
$ git clone urlremoterepository

- secara default, clone akan membuat project dengan nama folder sama dengan nama project
remote repository
- jika kita ingin melakukan clone, dengan nama folder yang berbeda dengan project remote
repository, kita bisa gunakan perintah:
$ git clone urlremoterepository namafolder

# Default hasil clone
- default clone akan berisi remote repository origin ke git remote repository yang kita clone
- default clone akan berisikan branch utama di remote repository
```
