### Git Revert Commit

```sh
$ git revert <hash>

1. fitur yang membatalkan commit yang sudah kita lakukan dengan cara
membuat commit baru yang membatalkan commit sebelumnya.

2. misal kita sudah melakukan commit data perubahan dari text eko menjadi eka, jika kita revert
secara otomatis akan membuat commit baru dengan melakukan perubahan dari eko ke eko.
```
